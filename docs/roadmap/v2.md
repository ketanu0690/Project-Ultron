# v2 Specification

## Purpose

Define **version 2.0** — the complete ULTRON AI WORLD experience with full scale navigation, governance, and thousands of agents.

---

## v2 Goal

> A user can travel from galaxy to agent memory through a seamless 3D experience, govern the AI civilization, and observe 5,000 agents in a living simulation.

---

### Swarm Rendering

At 5,000 agents, individual avatars are only rendered for:

- Agents in current room (full avatar)
- Agents in current district (LOD avatar)
- Agents elsewhere (dot on mini-map only)
